¿Qué es el manejo de archivos?

El manejo de archivos permite guardar y leer información desde archivos externos, como archivos de texto o CSV.

Librerías utilizadas


import java.io.BufferedReader;
import java.io.FileReader;

Lectura de un archivo CSV


BufferedReader br =
new BufferedReader(
new FileReader("datos.csv"));

Ejercicio resuelto

Leer un archivo CSV e imprimir su contenido. 


while((linea = br.readLine()) != null){
    System.out.println(linea);
}
