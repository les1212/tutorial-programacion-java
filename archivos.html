¿Qué es el manejo de archivos?

El manejo de archivos permite almacenar información de forma permanente en el computador. Gracias a ello, los datos pueden conservarse incluso después de que el programa termine su ejecución.

En Java es común trabajar con archivos de texto (.txt) y archivos CSV (.csv), los cuales permiten guardar información organizada para posteriormente leerla, modificarla o analizarla.

¿Para qué sirve?

  • Guardar información de usuarios.
  • Almacenar registros de ventas.
  • Leer datos desde archivos CSV.
  • Generar reportes.
  • Mantener información después de cerrar el programa.

Librerías utilizadas

Para leer archivos en Java se utilizan las siguientes librerías: 


import java.io.BufferedReader;
import java.io.FileReader;
  • FileReader: Permite abrir un archivo para leerlo.
  • BufferedReader: Facilita la lectura línea por línea.

Apertura de un archivo

Primero se debe indicar la ruta o nombre del archivo que se desea leer. 


String archivo = "datos.csv";

Luego se crea el objeto BufferedReader para acceder al contenido. 


BufferedReader br =
new BufferedReader(
new FileReader(archivo));

Lectura línea por línea

La lectura se realiza utilizando un ciclo while. El programa continuará leyendo mientras existan líneas disponibles. 


String linea;

while((linea = br.readLine()) != null){
    System.out.println(linea);
}

Ejemplo explicado

El siguiente programa abre un archivo CSV y muestra todo su contenido en pantalla. 


import java.io.BufferedReader;
import java.io.FileReader;

public class ManejoArchivos {

    public static void main(String[] args) {

        String archivo = "datos.csv";

        try {

            BufferedReader br =
                new BufferedReader(
                new FileReader(archivo));

            String linea;

            while((linea = br.readLine()) != null) {
                System.out.println(linea);
            }

            br.close();

        } catch(Exception e) {

            System.out.println("Error: " + e.getMessage());

        }
    }
}

Explicación del código

  • Se define el nombre del archivo a leer.
  • Se crea un FileReader para abrir el archivo.
  • BufferedReader permite leer línea por línea.
  • El ciclo while recorre todo el archivo.
  • Cuando no existen más líneas, la lectura termina.
  • El bloque catch captura errores como archivos inexistentes.

Conclusión

El manejo de archivos es una herramienta fundamental en programación, ya que permite almacenar y recuperar información de manera permanente. En Java, las clases FileReader y BufferedReader facilitan la lectura de archivos de texto y CSV de forma sencilla y eficiente.
