Manejo de Archivos en Java

¿Qué es el manejo de archivos?

El manejo de archivos permite guardar y recuperar información desde archivos externos. Esto es importante porque los datos no se pierden cuando el programa termina.

Idea principal: Un archivo permite almacenar datos de forma permanente. Por ejemplo, nombres de usuarios, productos, ventas, calificaciones o registros.

En Java se puede trabajar con archivos de texto como .txt y archivos separados por comas como .csv. Los archivos CSV son muy utilizados porque permiten guardar datos en forma de tabla.

Ejemplo de archivo CSV

Un archivo llamado datos.csv podría tener esta información: 


Nombre,Edad,Ciudad
Ana,20,Quito
Luis,22,Ambato
María,19,Guayaquil

Cada línea representa un registro y cada dato está separado por comas.

¿Para qué sirve?

  • Guardar información de usuarios.
  • Almacenar registros de ventas.
  • Leer datos desde archivos CSV.
  • Generar reportes.
  • Guardar récords o resultados de un programa.
  • Mantener información después de cerrar el programa.

Aplicaciones reales

  • Un sistema de facturación puede guardar productos y clientes en archivos CSV.
  • Un juego puede guardar el nombre del jugador y su puntaje.
  • Un sistema académico puede leer notas de estudiantes desde un archivo.
  • Un inventario puede cargar productos, precios y stock desde un archivo.

Librerías utilizadas

Para leer archivos en Java se utilizan clases del paquete java.io. 


import java.io.BufferedReader;
import java.io.FileReader;
  • FileReader: permite abrir un archivo para leerlo.
  • BufferedReader: permite leer el archivo línea por línea.
Importante: FileReader abre el archivo, mientras que BufferedReader ayuda a leerlo de forma más ordenada y eficiente.

Apertura de un archivo

Primero se debe indicar el nombre o ruta del archivo. 


String archivo = "datos.csv";

Después se crea el lector del archivo. 


BufferedReader br =
    new BufferedReader(
    new FileReader(archivo));

Lectura línea por línea

La lectura se realiza con un ciclo while. El método readLine() lee una línea del archivo. Cuando ya no existen más líneas, devuelve null. 


String linea;

while((linea = br.readLine()) != null){
    System.out.println(linea);
}

Análisis línea por línea

  • String linea; crea una variable para guardar cada línea del archivo.
  • br.readLine() lee una línea del archivo.
  • != null verifica que todavía existan líneas para leer.
  • System.out.println(linea); muestra la línea en pantalla.

Ejemplo explicado

El siguiente programa abre un archivo CSV llamado datos.csv y muestra todo su contenido en pantalla. 


import java.io.BufferedReader;
import java.io.FileReader;

public class ManejoArchivos {

    public static void main(String[] args) {

        String archivo = "datos.csv";

        try {

            BufferedReader br =
                new BufferedReader(
                new FileReader(archivo));

            String linea;

            while((linea = br.readLine()) != null) {
                System.out.println(linea);
            }

            br.close();

        } catch(Exception e) {

            System.out.println("Error: " + e.getMessage());

        }
    }
}

Explicación del código

  • Se importan las librerías necesarias para leer archivos.
  • Se define el nombre del archivo que se desea leer.
  • El bloque try contiene el código que puede generar errores.
  • FileReader abre el archivo.
  • BufferedReader permite leer el archivo línea por línea.
  • El ciclo while se repite mientras existan líneas disponibles.
  • br.close() cierra el archivo después de usarlo.
  • El bloque catch captura errores, por ejemplo si el archivo no existe.

Errores comunes

  • Escribir mal el nombre del archivo.
  • Guardar el archivo en una carpeta diferente a la del proyecto.
  • No cerrar el archivo con br.close().
  • Olvidar importar BufferedReader y FileReader.
  • Usar una ruta con barras invertidas sin cuidado, por ejemplo C:\datos\archivo.csv.
Consejo: Para evitar errores de ruta, al inicio es mejor colocar el archivo CSV dentro de la carpeta principal del proyecto.

Ejercicio práctico

Desarrollar un programa que lea un archivo CSV llamado datos.csv y muestre todas sus líneas en pantalla.

Solución completa


import java.io.BufferedReader;
import java.io.FileReader;

public class LeerArchivo {

    public static void main(String[] args) {

        String archivo = "datos.csv";

        try {

            BufferedReader br =
                new BufferedReader(
                new FileReader(archivo));

            String linea;

            while((linea = br.readLine()) != null) {

                System.out.println(linea);

            }

            br.close();

        } catch(Exception e) {

            System.out.println("Error: " + e.getMessage());

        }
    }
}

Explicación de la solución

  • El programa intenta abrir el archivo datos.csv.
  • Si el archivo existe, se lee línea por línea.
  • Cada línea se muestra en consola.
  • Cuando termina la lectura, se cierra el archivo.
  • Si ocurre un error, el programa muestra un mensaje explicando el problema.

Preguntas frecuentes

¿Qué pasa si el archivo no existe?

El programa entra al bloque catch y muestra un mensaje de error.

¿Qué significa CSV?

CSV significa valores separados por comas. Es un formato usado para guardar datos en forma de tabla.

¿Puedo leer archivos .txt?

Sí. El mismo procedimiento sirve para leer archivos de texto, cambiando el nombre del archivo, por ejemplo datos.txt.

Relación con otros temas

El manejo de archivos se relaciona con listas, vectores y matrices porque muchas veces los datos leídos desde un archivo se almacenan en esas estructuras para procesarlos después.

Por ejemplo, se puede leer un archivo CSV de ventas y guardar sus datos en una matriz para calcular totales, promedios o estadísticas.

Resumen

  • Los archivos permiten guardar información de forma permanente.
  • Java puede leer archivos usando FileReader y BufferedReader.
  • El método readLine() permite leer una línea a la vez.
  • Los archivos CSV guardan datos separados por comas.
  • El bloque try-catch permite manejar errores durante la lectura.

Conclusión

El manejo de archivos es una herramienta fundamental en programación porque permite trabajar con información real almacenada fuera del programa. En Java, las clases FileReader y BufferedReader facilitan la lectura de archivos de texto y CSV de una forma sencilla, permitiendo desarrollar programas más útiles y completos.
