Asignatura

Fundamentos de Programación

Objetivo General

Desarrollar un tutorial educativo que sirva como material de consulta y estudio para estudiantes interesados en aprender programación en Java, mediante explicaciones claras, ejemplos prácticos y ejercicios resueltos.

Objetivo

Este tutorial tiene como objetivo explicar de manera clara y sencilla los temas principales de programación en Java.

Temas del tutorial

  • Ciclos o bucles
  • Métodos o funciones
  • Vectores y matrices
  • Listas (ArrayList)
  • Manejo de archivos

Descripción

Cada tema contiene una explicación teórica, ejemplos de código, ejercicios prácticos resueltos y enlaces al código fuente alojado en GitHub para facilitar el aprendizaje.

¿Qué aprenderás?

  • Utilizar ciclos para repetir instrucciones.
  • Crear y utilizar métodos para organizar programas.
  • Trabajar con vectores y matrices.
  • Gestionar colecciones mediante ArrayList.
  • Leer archivos de texto y CSV en Java.
