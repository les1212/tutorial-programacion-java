¿Qué son las listas?

Las listas son estructuras de datos que permiten almacenar varios elementos de un mismo tipo. A diferencia de los vectores, las listas pueden aumentar o disminuir su tamaño durante la ejecución del programa.

Importante: Un ArrayList es útil cuando no sabemos cuántos datos vamos a guardar, porque su tamaño puede crecer automáticamente.

En Java, una de las listas más utilizadas es ArrayList, que pertenece al paquete java.util.

¿Para qué sirven?

  • Guardar colecciones de datos.
  • Almacenar información cuyo tamaño puede variar.
  • Agregar y eliminar elementos fácilmente.
  • Recorrer datos usando ciclos.
  • Gestionar usuarios, productos, ventas o registros.

Aplicaciones reales

  • Lista de productos en un sistema de facturación.
  • Lista de estudiantes en un sistema académico.
  • Lista de puntajes en un juego.
  • Datos leídos desde un archivo CSV.

Importación de la librería

Antes de usar ArrayList, debes importar la clase desde el paquete java.util: 


// Importamos la clase ArrayList para usar listas dinámicas
import java.util.ArrayList;

Creación de una lista

Para crear una lista que almacena cadenas de texto: 


// Creamos una lista vacía llamada 'nombres' que guardará textos (String)
ArrayList<String> nombres = new ArrayList<>();

Nota: <String> es el tipo de dato que la lista almacenará. Puede ser Integer, Double, Boolean, etc.

Ejemplo básico: ArrayList de números

También podemos crear una lista de números enteros: 


// Importamos la librería necesaria
import java.util.ArrayList;

public class ListaNumeros {
    public static void main(String[] args) {
        // Creamos una lista de números enteros
        ArrayList<Integer> numeros = new ArrayList<>();

        // Agregamos números a la lista
        numeros.add(10);
        numeros.add(20);
        numeros.add(30);

        // Recorremos la lista y mostramos cada número
        for (int n : numeros) {
            System.out.println(n);  // Muestra 10, 20, 30
        }
    }
}

Métodos más utilizados

Método Función Ejemplo
add() Agrega un elemento al final de la lista. lista.add("Ana")
get() Obtiene un elemento según su posición (índice). lista.get(0)
set() Modifica un elemento en una posición específica. lista.set(0, "Luis")
remove() Elimina un elemento por índice o por valor. lista.remove(1)
size() Devuelve la cantidad de elementos en la lista. lista.size()
clear() Vacía todos los elementos de la lista. lista.clear()
contains() Verifica si la lista contiene un elemento. lista.contains("Ana")

Agregar y obtener elementos

Agregamos elementos con add() y los leemos con get(): 


// Creamos la lista de nombres
ArrayList<String> nombres = new ArrayList<>();

// Agregamos elementos uno por uno
nombres.add("Ana");   // Posición 0
nombres.add("Luis");  // Posición 1
nombres.add("María"); // Posición 2

// Mostramos el primer elemento (índice 0)
System.out.println(nombres.get(0)); // Imprime: Ana

Recorrer una lista

Se puede recorrer con un ciclo for mejorado (for-each) o con un for tradicional: 


// For-each: recorre cada elemento directamente
for (String nombre : nombres) {
    System.out.println(nombre);
}

// For tradicional: usamos el índice y el tamaño de la lista
for (int i = 0; i < nombres.size(); i++) {
    System.out.println(nombres.get(i));
}

Análisis línea por línea

  • ArrayList<String> indica que la lista guardará textos.
  • nombres es el nombre de la lista.
  • new ArrayList<>() crea la lista vacía.
  • add() agrega elementos al final.
  • get() lee un elemento por su posición.
  • size() devuelve el número de elementos.
  • El ciclo for recorre todos los datos almacenados.

Ejemplo teórico: lista de productos

Este ejemplo muestra cómo crear una lista, agregar elementos y recorrerla con un ciclo for-each: 


// Importamos la clase ArrayList
import java.util.ArrayList;

// Clase principal
public class ListasJava {

    public static void main(String[] args) {

        // Creamos una lista de productos (textos)
        ArrayList<String> productos = new ArrayList<>();

        // Agregamos productos a la lista
        productos.add("Arroz");   // Posición 0
        productos.add("Leche");   // Posición 1
        productos.add("Pan");     // Posición 2
        productos.add("Azúcar");  // Posición 3

        // Recorremos la lista con ciclo for-each
        for (String producto : productos) {
            // Mostramos cada producto en pantalla
            System.out.println(producto);
        }
    }
}

Explicación del código

  • Se importa ArrayList del paquete java.util.
  • Se crea una lista llamada productos que almacena textos.
  • Se agregan cuatro productos con add().
  • Se recorre la lista con un ciclo for mejorado.
  • Cada producto se muestra en pantalla.

Diferencia entre Vector y ArrayList

Vector ArrayList
Tamaño fijo Tamaño dinámico
Se define su tamaño al crearlo Crece automáticamente
Útil cuando se conoce la cantidad de datos Útil cuando la cantidad de datos puede cambiar
No puede agregar más allá del límite Puede agregar sin límite (memoria permitiendo)

Errores comunes

  • Olvidar importar java.util.ArrayList.
  • Intentar acceder a una posición que no existe (índice fuera de rango).
  • Confundir size() con length (los vectores usan length, las listas usan size()).
  • Olvidar indicar el tipo de dato de la lista (<String>, <Integer>, etc.).
  • Usar tipos primitivos (como int) en lugar de clases envoltorio (Integer).

Ejercicio práctico

Crear una lista con los nombres de cinco estudiantes y mostrarlos en pantalla.

Solución completa


// Importamos la clase ArrayList
import java.util.ArrayList;

// Clase principal del ejercicio
public class ListaEstudiantes {

    public static void main(String[] args) {

        // Creamos una lista de estudiantes (tipo String)
        ArrayList<String> estudiantes = new ArrayList<>();

        // Agregamos cinco nombres a la lista
        estudiantes.add("Juan");    // Posición 0
        estudiantes.add("María");   // Posición 1
        estudiantes.add("Pedro");   // Posición 2
        estudiantes.add("Ana");     // Posición 3
        estudiantes.add("Carlos");  // Posición 4

        // Recorremos la lista con ciclo for-each
        for (String estudiante : estudiantes) {
            // Mostramos cada nombre en pantalla
            System.out.println(estudiante);
        }
    }
}

Explicación de la solución

  • Se crea una lista llamada estudiantes de tipo String.
  • Se agregan cinco nombres con add().
  • El ciclo for recorre todos los nombres.
  • Cada nombre se muestra en pantalla.

Ejercicio adicional: eliminar y modificar elementos

Ahora vamos a crear una lista, modificar un elemento, eliminar otro y mostrar el resultado final. 


// Importamos la librería
import java.util.ArrayList;

public class ModificarLista {
    public static void main(String[] args) {

        // Creamos la lista de colores
        ArrayList<String> colores = new ArrayList<>();

        // Agregamos colores
        colores.add("Rojo");      // Posición 0
        colores.add("Verde");     // Posición 1
        colores.add("Azul");      // Posición 2
        colores.add("Amarillo");  // Posición 3

        // Modificamos el color en la posición 1
        colores.set(1, "Naranja"); // Ahora: Rojo, Naranja, Azul, Amarillo

        // Eliminamos el color en la posición 2 (Azul)
        colores.remove(2);           // Ahora: Rojo, Naranja, Amarillo

        // Mostramos la lista final
        System.out.println("Colores finales:");
        for (String color : colores) {
            System.out.println(color);
        }

        // Mostramos el tamaño de la lista
        System.out.println("Total de colores: " + colores.size());
    }
}

Preguntas frecuentes

¿ArrayList es igual que un vector?

No. Un vector tiene tamaño fijo, mientras que un ArrayList puede cambiar de tamaño dinámicamente.

¿Qué uso para saber cuántos elementos tiene?

Se usa el método size().

¿Puedo guardar números en un ArrayList?

Sí, pero debes usar Integer (no int), por ejemplo: ArrayList<Integer>.

¿Puedo mezclar tipos de datos en una lista?

No. Un ArrayList solo puede guardar un tipo de dato. Si necesitas mezclar, usa una lista de Object (no recomendado para principiantes).

Relación con otros temas

Las listas se relacionan con ciclos porque se recorren mediante bucles. También se relacionan con archivos, ya que muchas veces los datos leídos desde un CSV se guardan en un ArrayList para procesarlos después.

Resumen

  • ArrayList permite almacenar varios datos de un mismo tipo.
  • Su tamaño puede cambiar dinámicamente (agregar o eliminar).
  • Se usa add() para agregar datos.
  • Se usa get() para leer un elemento por posición.
  • Se usa set() para modificar un elemento.
  • Se usa remove() para eliminar un elemento.
  • Se usa size() para conocer la cantidad de elementos.
  • Se puede recorrer con ciclos for o for-each.

Conclusión

Las listas son estructuras de datos muy útiles porque permiten almacenar cantidades variables de información. La clase ArrayList facilita la gestión de datos dinámicos y es ampliamente utilizada en programas reales para manejar usuarios, productos, registros y datos provenientes de archivos.
