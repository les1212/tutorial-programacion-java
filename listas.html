¿Qué son las listas?

Las listas son estructuras de datos que permiten almacenar varios elementos de un mismo tipo. A diferencia de los vectores, las listas pueden aumentar o disminuir su tamaño durante la ejecución del programa.

Importante: Un ArrayList es útil cuando no sabemos cuántos datos vamos a guardar, porque su tamaño puede crecer automáticamente.

En Java, una de las listas más utilizadas es ArrayList, que pertenece al paquete java.util.

¿Para qué sirven?

  • Guardar colecciones de datos.
  • Almacenar información cuyo tamaño puede variar.
  • Agregar y eliminar elementos fácilmente.
  • Recorrer datos usando ciclos.
  • Gestionar usuarios, productos, ventas o registros.

Aplicaciones reales

  • Lista de productos en un sistema de facturación.
  • Lista de estudiantes en un sistema académico.
  • Lista de puntajes en un juego.
  • Datos leídos desde un archivo CSV.

Importación de la librería


import java.util.ArrayList;

Creación de una lista


ArrayList<String> nombres = new ArrayList<>();

En este caso, la lista almacenará datos de tipo String.

Métodos más utilizados

Método Función
add() Agrega un elemento a la lista.
get() Obtiene un elemento según su posición.
remove() Elimina un elemento.
size() Devuelve el tamaño de la lista.

Agregar elementos


nombres.add("Ana");
nombres.add("Luis");
nombres.add("María");

Recorrer una lista


for(String nombre : nombres){
    System.out.println(nombre);
}

Análisis línea por línea

  • ArrayList<String> indica que la lista guardará textos.
  • nombres es el nombre de la lista.
  • new ArrayList<>() crea la lista vacía.
  • add() agrega elementos.
  • El ciclo for recorre todos los datos almacenados.

Ejemplo explicado


import java.util.ArrayList;

public class ListasJava {

    public static void main(String[] args) {

        ArrayList<String> productos = new ArrayList<>();

        productos.add("Arroz");
        productos.add("Leche");
        productos.add("Pan");
        productos.add("Azúcar");

        for(String producto : productos){
            System.out.println(producto);
        }
    }
}

Explicación del código

  • Se importa ArrayList.
  • Se crea una lista llamada productos.
  • Se agregan cuatro productos con add().
  • Se recorre la lista con un ciclo for mejorado.
  • Cada producto se muestra en pantalla.

Diferencia entre Vector y ArrayList

Vector ArrayList
Tamaño fijo Tamaño dinámico
Se define su tamaño al crearlo Crece automáticamente
Útil cuando se conoce la cantidad de datos Útil cuando la cantidad de datos puede cambiar

Errores comunes

  • Olvidar importar java.util.ArrayList.
  • Intentar acceder a una posición que no existe.
  • Confundir size() con length.
  • Olvidar indicar el tipo de dato de la lista.

Ejercicio práctico

Crear una lista con los nombres de cinco estudiantes y mostrarlos en pantalla.

Solución completa


import java.util.ArrayList;

public class ListaEstudiantes {

    public static void main(String[] args) {

        ArrayList estudiantes = new ArrayList<>();

        estudiantes.add("Juan");
        estudiantes.add("María");
        estudiantes.add("Pedro");
        estudiantes.add("Ana");
        estudiantes.add("Carlos");

        for(String estudiante : estudiantes) {
            System.out.println(estudiante);
        }
    }
}

Explicación de la solución

  • Se crea una lista llamada estudiantes.
  • Se agregan cinco nombres con add().
  • El ciclo for recorre todos los nombres.
  • Cada nombre se muestra en pantalla.

Preguntas frecuentes

¿ArrayList es igual que un vector?

No. Un vector tiene tamaño fijo, mientras que un ArrayList puede cambiar de tamaño.

¿Qué uso para saber cuántos elementos tiene?

Se usa el método size().

Relación con otros temas

Las listas se relacionan con ciclos porque se recorren mediante bucles. También se relacionan con archivos, ya que muchas veces los datos leídos desde un CSV se guardan en un ArrayList.

Resumen

  • ArrayList permite almacenar varios datos.
  • Su tamaño puede cambiar dinámicamente.
  • Se usa add() para agregar datos.
  • Se usa size() para conocer la cantidad de elementos.
  • Se puede recorrer con ciclos.

Conclusión

Las listas son estructuras de datos muy útiles porque permiten almacenar cantidades variables de información. La clase ArrayList facilita la gestión de datos dinámicos y es ampliamente utilizada en programas reales para manejar usuarios, productos, registros y datos provenientes de archivos.
