¿Qué son las listas?

Las listas son estructuras de datos que permiten almacenar varios elementos de un mismo tipo. A diferencia de los vectores, las listas pueden aumentar o disminuir su tamaño dinámicamente durante la ejecución del programa.

En Java, una de las implementaciones más utilizadas es la clase ArrayList, perteneciente al paquete java.util.

¿Para qué sirven?

  • Guardar colecciones de datos.
  • Almacenar información cuyo tamaño puede variar.
  • Agregar y eliminar elementos fácilmente.
  • Recorrer información mediante ciclos.
  • Gestionar registros de usuarios, productos o ventas.

Importación de la librería

Antes de utilizar una lista es necesario importar la clase ArrayList. 


import java.util.ArrayList;

Creación de una lista

Una lista se crea indicando el tipo de dato que almacenará. 


ArrayList nombres = new ArrayList<>();

En este caso la lista almacenará cadenas de texto (String).

Agregar elementos

El método add() permite insertar nuevos elementos. 


nombres.add("Ana");
nombres.add("Luis");
nombres.add("María");

Recorrer una lista

Para mostrar todos los elementos se puede utilizar un ciclo for mejorado. 


for(String nombre : nombres){
    System.out.println(nombre);
}

Ejemplo explicado

El siguiente programa crea una lista de productos y posteriormente muestra todos sus elementos. 


import java.util.ArrayList;

public class ListasJava {

    public static void main(String[] args) {

        ArrayList productos = new ArrayList<>();

        productos.add("Arroz");
        productos.add("Leche");
        productos.add("Pan");
        productos.add("Azúcar");

        for(String producto : productos){
            System.out.println(producto);
        }
    }
}

Explicación del código

  • Se crea una lista llamada productos.
  • Se agregan cuatro elementos mediante add().
  • El ciclo for recorre todos los elementos almacenados.
  • Cada producto es mostrado en pantalla.

Ejercicio práctico

Crear una lista con los nombres de cinco estudiantes y mostrarlos en pantalla.

Solución


ArrayList estudiantes = new ArrayList<>();

estudiantes.add("Juan");
estudiantes.add("María");
estudiantes.add("Pedro");
estudiantes.add("Ana");
estudiantes.add("Carlos");

for(String estudiante : estudiantes){
    System.out.println(estudiante);
}

Conclusión

Las listas son estructuras de datos muy útiles porque permiten almacenar cantidades variables de información. La clase ArrayList facilita la gestión de datos dinámicos y es ampliamente utilizada en aplicaciones reales para manejar usuarios, productos, registros y muchos otros tipos de información.
