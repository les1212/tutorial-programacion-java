¿Qué son los métodos?

Los métodos son bloques de código que realizan una tarea específica dentro de un programa. Su principal objetivo es organizar mejor el código y evitar repetir instrucciones innecesariamente.

Importante: Los métodos permiten dividir un programa en tareas más pequeñas, haciendo que el código sea más fácil de entender, reutilizar y mantener.

¿Para qué sirven?

  • Organizar mejor los programas.
  • Reutilizar código.
  • Reducir la repetición de instrucciones.
  • Facilitar el mantenimiento de un programa.
  • Hacer que el código sea más legible.

Aplicaciones reales

  • Calcular promedios de estudiantes.
  • Validar datos ingresados por el usuario.
  • Procesar ventas y facturas.
  • Realizar operaciones matemáticas repetitivas.
  • Organizar programas grandes en partes más pequeñas.

Estructura de un método

La estructura general de un método en Java es la siguiente: 


public static tipoRetorno nombreMetodo(parametros){

    instrucciones;

}

Cada parte cumple una función:

  • public: Indica que el método puede ser utilizado desde otras clases.
  • static: Permite llamar al método sin crear un objeto.
  • tipoRetorno: Es el tipo de dato que devuelve el método.
  • nombreMetodo: Nombre que identifica al método.
  • parámetros: Datos que recibe el método para trabajar.

Ejemplo sencillo

El siguiente método muestra un mensaje en pantalla: 


public static void saludar() {

    System.out.println("Bienvenido al programa");

}

Para ejecutar el método se debe llamar desde el programa principal: 


saludar();

Métodos con retorno

Un método con retorno devuelve un valor mediante la palabra reservada return. 


public static int sumar(int a, int b){

    return a + b;

}

En este ejemplo el método recibe dos números enteros y devuelve su suma.

Métodos sin retorno

Cuando un método no devuelve ningún valor se utiliza la palabra void. 


public static void saludar(){

    System.out.println("Hola");

}

Ejemplo explicado

El siguiente programa utiliza un método para sumar dos números. 


public class Metodos {

    public static int sumar(int a, int b){

        return a + b;

    }

    public static void main(String[] args){

        int resultado = sumar(10,5);

        System.out.println("La suma es: " + resultado);

    }
}

Análisis línea por línea

  • public → permite acceder al método desde otras clases.
  • static → permite llamar al método sin crear un objeto.
  • int → indica que el método devuelve un número entero.
  • sumar → es el nombre del método.
  • (int a, int b) → son los parámetros que recibe.
  • return → devuelve el resultado de la operación.

Explicación del código

  • Se crea el método sumar().
  • El método recibe dos parámetros enteros.
  • La instrucción return devuelve la suma.
  • Desde main() se llama al método.
  • El resultado se almacena en una variable.
  • Finalmente se imprime el resultado.

Ventajas de utilizar métodos

  • Evitan escribir el mismo código varias veces.
  • Facilitan la corrección de errores.
  • Permiten reutilizar funciones en diferentes partes del programa.
  • Hacen que el código sea más profesional y organizado.

Métodos con parámetros y sin parámetros

Un método puede recibir datos mediante parámetros o trabajar sin ellos.

Método sin parámetros


public static void mostrarMensaje() {

    System.out.println("Hola");

}

Método con parámetros


public static void saludar(String nombre) {

    System.out.println("Hola " + nombre);

}

Errores comunes

  • Olvidar escribir la palabra return en métodos con retorno.
  • Confundir métodos void con métodos que devuelven valores.
  • Llamar al método con parámetros incorrectos.
  • Escribir mal el nombre del método.

Ejercicio práctico

Crear un método que reciba dos números enteros y devuelva el número mayor. Posteriormente mostrar el resultado en pantalla.

Solución completa


public class MetodoMayor {

    public static int mayor(int a, int b) {

        if(a > b) {
            return a;
        } else {
            return b;
        }

    }

    public static void main(String[] args) {

        int resultado = mayor(15, 8);

        System.out.println("El número mayor es: " + resultado);

    }
}

Explicación de la solución

  • Se comparan los dos números.
  • Si el primero es mayor, se devuelve ese valor.
  • En caso contrario se devuelve el segundo número.

Preguntas frecuentes

¿Qué diferencia existe entre void e int?

void no devuelve ningún valor, mientras que int devuelve un número entero.

¿Qué son los parámetros?

Son datos que recibe un método para realizar una tarea específica.

Conclusión

Los métodos son una de las herramientas más importantes de la programación. Permiten dividir los programas en tareas más pequeñas, reutilizar código y mejorar la organización de las aplicaciones. Su uso es fundamental para el desarrollo de programas más eficientes y fáciles de mantener.
