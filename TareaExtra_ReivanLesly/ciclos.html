¿Qué son los ciclos?

Los ciclos permiten repetir instrucciones varias veces. En Java se usan principalmente los ciclos for, while y do while.

Ejemplo


for (int i = 1; i <= 10; i++) {
    System.out.println(i);
}

Ejercicio resuelto

Mostrar los números pares del 1 al 20.


public class Ciclos {
    public static void main(String[] args) {
        for (int i = 1; i <= 20; i++) {
            if (i % 2 == 0) {
                System.out.println(i);
            }
        }
    }
}
