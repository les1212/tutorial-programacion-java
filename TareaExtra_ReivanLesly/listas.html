¿Qué es una lista?

Una lista permite almacenar varios datos y cambiar su tamaño dinámicamente. En Java se puede usar ArrayList para trabajar con listas.

Importar ArrayList


import java.util.ArrayList;

Ejemplo


ArrayList<String> nombres = new ArrayList<>();

nombres.add("Ana");
nombres.add("Luis");
nombres.add("María");

for(String nombre : nombres){
    System.out.println(nombre);
}

Ejercicio resuelto

Crear una lista de productos y mostrar todos sus elementos. 


ArrayList<String> productos = new ArrayList<>();

productos.add("Arroz");
productos.add("Leche");
productos.add("Pan");

for(String producto : productos){
    System.out.println(producto);
}
