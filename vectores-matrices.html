Estructuras para almacenar y procesar varios datos

¿Qué son los vectores?

Un vector es una estructura de datos unidimensional que permite almacenar varios elementos del mismo tipo usando una sola variable.

Importante: Un vector permite guardar varios valores en una misma variable, pero cada valor se identifica por una posición llamada índice.

Los índices comienzan desde la posición 0: 


Índice:   0    1    2    3    4
Vector: [10] [20] [30] [40] [50]


// Declaramos un vector de números enteros con 5 elementos
int[] numeros = {10, 20, 30, 40, 50};

¿Para qué sirven los vectores?

  • Guardar notas de estudiantes.
  • Registrar temperaturas de cada día.
  • Almacenar ventas diarias o semanales.
  • Realizar sumas, promedios, máximos y mínimos.
  • Ordenar listas de números.

Recorrido de un vector

Para recorrer un vector, usamos un ciclo for que avanza desde la posición 0 hasta la última posición del vector. 


// Recorremos el vector desde la posición 0 hasta el final
for(int i = 0; i < numeros.length; i++) {
    // Mostramos cada elemento usando su índice
    System.out.println(numeros[i]);
}

Análisis línea por línea

  • int i = 0 inicia el recorrido en la primera posición.
  • i < numeros.length evita salir del tamaño del vector.
  • i++ avanza a la siguiente posición.
  • numeros[i] accede al elemento actual.
  • numeros.length devuelve la cantidad total de elementos.

Vector vacío: crear sin valores iniciales

También podemos crear un vector sin datos al inicio, especificando solo el tamaño. Luego asignamos valores posición por posición. 


// Creamos un vector vacío de 4 espacios para números decimales
double[] temperaturas = new double[4];

// Asignamos un valor a cada posición manualmente
temperaturas[0] = 22.5;  // Primera posición
temperaturas[1] = 24.0;  // Segunda posición
temperaturas[2] = 19.8;  // Tercera posición
temperaturas[3] = 21.0;  // Cuarta posición

¿Qué son las matrices?

Una matriz es una estructura bidimensional formada por filas y columnas. Se puede imaginar como una tabla o una hoja de cálculo.

Importante: En una matriz se usan dos índices: uno para la fila y otro para la columna. 

// Declaramos una matriz de 3 filas y 3 columnas
int[][] matriz = {
    {1, 2, 3},   // Fila 0
    {4, 5, 6},   // Fila 1
    {7, 8, 9}    // Fila 2
};

Visualmente: 


Fila 0:  1  2  3
Fila 1:  4  5  6
Fila 2:  7  8  9

Tabla comparativa: vector vs. matriz

Característica Vector Matriz
Dimensiones 1 (solo una fila) 2 (filas y columnas)
Índices Uno: [i] Dos: [fila][columna]
Ciclos para recorrer 1 ciclo for 2 ciclos for anidados
Ejemplo real Notas de un solo estudiante Notas de varios estudiantes por materia

¿Para qué sirven las matrices?

  • Representar tablas de datos.
  • Almacenar calificaciones de múltiples estudiantes.
  • Gestionar ventas por sucursal y por producto.
  • Trabajar con imágenes digitales (píxeles).
  • Resolver problemas con filas y columnas.

Recorrido de una matriz

Para recorrer una matriz se utilizan ciclos anidados. El primer ciclo recorre las filas y el segundo las columnas. 


// Recorremos cada fila de la matriz
for(int i = 0; i < matriz.length; i++) {

    // Recorremos cada columna de la fila actual
    for(int j = 0; j < matriz[i].length; j++) {

        // Mostramos el elemento en la fila i, columna j
        System.out.print(matriz[i][j] + " ");
    }

    // Saltamos a la siguiente línea después de cada fila
    System.out.println();
}

Análisis del recorrido de una matriz

  • i representa la fila actual.
  • j representa la columna actual.
  • matriz[i][j] accede a una posición específica.
  • El primer for controla las filas.
  • El segundo for controla las columnas.
  • matriz.length da la cantidad de filas.
  • matriz[i].length da la cantidad de columnas de la fila i.

Ejemplo explicado

El siguiente programa calcula la suma de los elementos de un vector y luego muestra una matriz en pantalla. 


public class VectoresMatrices {

    // Método principal donde inicia el programa
    public static void main(String[] args) {

        // Creamos un vector con cuatro números enteros
        int[] vector = {5, 10, 15, 20};

        // Variable para acumular la suma de los elementos
        int suma = 0;

        // Recorremos el vector para sumar cada elemento
        for(int i = 0; i < vector.length; i++) {
            // Acumulamos el valor actual en la variable suma
            suma += vector[i];
        }

        // Mostramos el resultado de la suma
        System.out.println("Suma = " + suma);

        // Creamos una matriz de 3 filas y 3 columnas
        int[][] matriz = {
            {1, 2, 3},
            {4, 5, 6},
            {7, 8, 9}
        };

        // Recorremos la matriz con ciclos anidados
        for(int i = 0; i < matriz.length; i++) {

            // Recorremos cada columna de la fila actual
            for(int j = 0; j < matriz[i].length; j++) {
                // Mostramos el elemento con espacio para separar
                System.out.print(matriz[i][j] + " ");
            }

            // Salto de línea después de cada fila
            System.out.println();
        }
    }
}

Explicación del código

  • Se crea un vector con cuatro números.
  • Se declara la variable suma para acumular valores.
  • El ciclo for recorre el vector completo.
  • Se suma cada valor usando suma += vector[i].
  • Después se crea una matriz de 3 filas y 3 columnas.
  • Los ciclos anidados recorren toda la matriz.
  • Se imprime cada fila en una línea diferente.

Aplicaciones reales

  • Notas de estudiantes por parcial.
  • Ventas por sucursal y por producto.
  • Horarios organizados por días y horas.
  • Imágenes representadas como matrices de píxeles.
  • Registro de temperaturas por ciudad y por día.
  • Asientos de un cine o avión organizados por filas y columnas.

Errores comunes

  • Intentar acceder a una posición que no existe (índice fuera de rango).
  • Olvidar que los índices empiezan en 0, no en 1.
  • Confundir filas con columnas al usar los índices.
  • Usar length sin paréntesis, ya que es una propiedad, no un método.
  • Olvidar usar ciclos anidados para recorrer matrices.
  • Intentar guardar datos de tipos diferentes en un mismo vector.

Ejercicio práctico: promedio de un vector

Desarrollar un programa que almacene cinco números en un vector y calcule su promedio.

Solución completa


public class PromedioVector {

    // Método principal donde inicia el programa
    public static void main(String[] args) {

        // Creamos un vector con cinco números enteros
        int[] numeros = {10, 20, 30, 40, 50};

        // Variable para acumular la suma de los elementos
        int suma = 0;

        // Recorremos el vector para sumar cada número
        for(int i = 0; i < numeros.length; i++) {
            // Sumamos el valor actual al acumulador
            suma += numeros[i];
        }

        // Calculamos el promedio convirtiendo suma a double
        double promedio = (double) suma / numeros.length;

        // Mostramos el resultado en pantalla
        System.out.println("Promedio = " + promedio);
    }
}

Explicación de la solución

  • Se crea un vector llamado numeros.
  • La variable suma inicia en 0.
  • El ciclo for recorre todos los elementos.
  • Cada número se acumula en la variable suma.
  • Se calcula el promedio dividiendo la suma entre la cantidad de elementos.
  • Se usa (double) para obtener un resultado con decimales.

Preguntas frecuentes

¿Cuál es la diferencia entre vector y matriz?

Un vector tiene una sola dimensión. Una matriz tiene dos dimensiones: filas y columnas.

¿Por qué los índices comienzan en 0?

Porque Java utiliza la posición 0 como el primer espacio de almacenamiento.

¿Qué significa length?

length devuelve la cantidad de elementos de un vector o la cantidad de filas de una matriz.

¿Puedo cambiar el tamaño de un vector después de crearlo?

No, el tamaño de un vector en Java es fijo. Si necesitas tamaño dinámico, usa una lista (ArrayList).

¿Puedo usar un vector para guardar textos?

Sí, usando String[] nombres = {"Ana", "Luis", "Carlos"};.

Relación con otros temas

Los vectores y matrices se relacionan directamente con ciclos, porque se necesita recorrer sus elementos. También se relacionan con métodos, ya que se pueden crear funciones para calcular sumas, promedios, máximos y mínimos. Además, se pueden combinar con archivos para leer o guardar datos en bloques.

Resumen

  • Los vectores almacenan datos en una sola dimensión.
  • Las matrices almacenan datos en filas y columnas.
  • Los índices empiezan en 0.
  • Los vectores se recorren con un ciclo for.
  • Las matrices se recorren con ciclos anidados.
  • El tamaño de un vector es fijo al momento de crearlo.

Conclusión

Los vectores y matrices son estructuras fundamentales en programación. Permiten almacenar, organizar y procesar grandes cantidades de información. Su uso es indispensable para resolver problemas con datos numéricos, registros, estadísticas, ventas e incluso procesamiento de imágenes.
