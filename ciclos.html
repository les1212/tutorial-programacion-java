¿Qué son los ciclos?

Los ciclos, también llamados bucles, son estructuras de control que permiten repetir un conjunto de instrucciones varias veces sin necesidad de escribir el mismo código repetidamente.

Imagina que deseas imprimir los números del 1 al 10. Sin un ciclo tendrías que escribir: 


System.out.println(1);
System.out.println(2);
System.out.println(3);
...
System.out.println(10);

Esto sería poco práctico. Los ciclos permiten automatizar estas tareas mediante una variable de control que cambia en cada repetición.

Tipos de ciclos en Java

  • for: Se utiliza cuando conocemos cuántas veces se repetirá una tarea.
  • while: Se utiliza cuando la repetición depende de una condición.
  • do while: Similar al while, pero garantiza al menos una ejecución.

Ciclo For

La estructura general de un ciclo for es: 


for(inicialización; condición; incremento){
    instrucciones;
}

Cada parte cumple una función:

  • Inicialización: Se ejecuta una sola vez al comenzar.
  • Condición: Determina si el ciclo continúa.
  • Incremento: Actualiza la variable de control.

Ejemplo explicado


for(int i = 1; i <= 10; i++) {
    System.out.println(i);
}

En este ejemplo:

  • La variable i inicia en 1.
  • Mientras i sea menor o igual a 10, el ciclo continúa.
  • Después de cada repetición, i aumenta en 1.

Salida: 


1
2
3
4

10

Ejercicio práctico

Desarrollar un programa que muestre todos los números pares del 1 al 20.

Solución


public class Ciclos {

    public static void main(String[] args) {

        for(int i = 1; i <= 20; i++) {

            if(i % 2 == 0) {

                System.out.println(i);

            }
        }
    }
}

Explicación de la solución

  • El ciclo recorre los números del 1 al 20.
  • El operador % calcula el residuo de una división.
  • Si un número dividido para 2 tiene residuo 0, significa que es par.
  • Solo los números pares son impresos en pantalla.

Conclusión

Los ciclos son herramientas fundamentales en programación porque permiten automatizar tareas repetitivas. El ciclo for es ideal cuando se conoce la cantidad de repeticiones, mientras que while y do while son útiles cuando la repetición depende de una condición.
